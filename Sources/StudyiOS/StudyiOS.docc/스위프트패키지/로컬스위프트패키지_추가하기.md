# 로컬 스위프트 패키지 추가하기

## 개요

배포하지 않고 로컬에서 작업한 스위프트 패키지를 사용하고 싶다면 아래 스크린샷을 따라 하면 된다.

> **TMI**: 나는 KuringMapKit 이라는 Swift Package 를 제공하는 kuring-mapkit-ios 라는 폴더를 추가할 것이다.
